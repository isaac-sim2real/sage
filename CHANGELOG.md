# Changelog

## [0.1.0] - 2025-10-22

### Initial Release
- Simulation execution in IsaacSim
- Real robot data collection via robot SDK
- Multi-metric analysis (RMSE, MAPE, Correlation, Cosine Similarity)
- Visualization tools (trajectory plots, boxplots, Excel reports)
- Support for Unitree H1-2, Unitree G1, Realman WR75S and AMASS dataset
